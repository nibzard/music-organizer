# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial PyPI package metadata preparation
- Single-file distribution for easy installation
- Auto-update mechanism
- GitHub Actions for automated releases

### Features
- Smart music classification (Albums, Live, Collaborations, Compilations, Rarities)
- Metadata enhancement with MusicBrainz lookup
- Interactive duplicate resolution
- Async and batch processing modes
- Plugin system for extensibility
- Dashboard for library visualization

## [0.1.0] - 2025-12-XX

### Added
- Initial release of music-organizer
- Core organization engine with metadata-aware categorization
- CLI with organize, scan, inspect, and validate commands
- Support for FLAC, MP3, WAV, M4A, and AAC formats
- Backup and rollback functionality
- Rich terminal output with progress bars
